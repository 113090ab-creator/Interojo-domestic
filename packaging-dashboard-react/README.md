# 국내제품 포장현황 대시보드 (React)

## 실행 방법

```bash
npm install
npm run dev
```

- 기본 주소: `http://127.0.0.1:8512`
- 포트가 이미 사용 중이면 `vite.config.js`에서 `server.port` 값을 변경하세요.

## 제공 기능 (1차 개발 범위)

- 요청물량 파일 + 포장실적 파일 업로드 (`CSV`, `XLSX`, `XLS`)
- 컬럼 자동 매핑 + 수동 매핑
- 제품코드 기준 요청/포장 집계
- 제품별 요청수량, 포장완료수량, 부족수량, 포장진행률 계산
- 상태 자동 분류
  - `미착수`: 요청수량 > 0, 포장수량 = 0
  - `부족`: 포장진행률 < 80%
  - `진행중`: 80% 이상 ~ 100% 미만
  - `완료`: 100% 이상
  - `초과`: 포장수량 > 요청수량
  - `요청外 포장`: 요청수량 = 0, 포장수량 > 0
- KPI 카드
  - 총 요청수량, 총 포장완료수량, 총 미포장수량
  - 전체 포장진행률, 부족 품목 수, 초과 포장 품목 수
- 메인 테이블
  - 정렬/검색/상태 필터
- 부족수량 우선순위 테이블
- 요청外 포장, 초과 포장 분리 표시
- 일자별 포장수량 추이 차트

## 데이터 매핑 팁

- 요청파일 필수: `제품코드`, `제품명`, `요청수량`
- 포장파일 필수:
  - `제품코드`, `제품명`
  - `포장수량` 또는 `팩수량 + 포장단위`
- 포장일자 매핑 시 일자별 추이 차트가 활성화됩니다.
