  - 우선 대응 품목

## 2) 계산 기준 정리
- 포장진도율:
  - `포장 PACK / 요청 PACK * 100`
  - 0~100% 범위로 표시.
- 포장부족수량:
  - `요청 PACK - 포장 PACK`
  - 부족수량은 오렌지 계열로 강조.
- 생산부족수량:
  - `수요정보(전공정)`의 `누수/규격검사 생산수량` 값을 생산부족수량으로 직접 반영.
  - 값이 없거나 0이면 생산부족수량 0으로 처리.
- 생산진도율:
  - `(요청 PCS - 생산부족수량) / 요청 PCS * 100`
  - 생산부족수량이 없으면 생산 완료로 보고 100% 처리.
- 상태:
  - 포장 기준으로만 판단.
  - `미착수`: 포장 PACK = 0
  - `진행중`: 포장 PACK > 0 and 포장진도율 < 100%
  - `완료`: 포장진도율 >= 100%

  - 본품 KPI

- 각 KPI 표시 항목:
  - 요청 PACK
  - 포장 PACK
  - 부족 PACK

- 본품 기준:
  - 전체에서 샘플을 제외한 모든 데이터.
- 샘플 기준:
  - 원본 제품명에 `샘플`이 포함된 데이터.
